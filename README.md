# C# WPF NotifyIcon MVVM DI Host sample

This is a basic sample project for the NotifyIcon component ([Hardcodet.NotifyIcon.Wpf](https://github.com/hardcodet/wpf-notifyicon)) together with 
MVVM ([CommunityToolkit.Mvvm](https://github.com/CommunityToolkit/MVVM-Samples)), DI (Microsoft.Extensions.DependencyInjection) and an 
application host (Microsoft.Extensions.Hosting) in .NET 10.0.
